# 引用

> **Reference**

## 定义

> 引用是已定义的变量的别名

```cpp
int a = 1;
cout << b; //输出1
```

---

## 左值与右值引用

>左值：能对表达式取地址、或具名对象/变量。一般指表达式结束后依然存在的持久对象。
>
>右值：不能对表达式取地址，或匿名对象。一般指表达式结束就不再存在的临时对象。

`int &&a = 5`*<——这就是右值引用的语法*

```cpp
void func(int &a)
{
    std::printf("%d ",a + 1);
}

void func(int &&a)
{
    std::printf("%d ",a + 2);
}
void main()
{
    int &&a = 1;
    func(a);
    func(1);
}
//func(a)为2，func(1)为3
//在重载上的应用
```
